## [1.0.0] - 2024

### Added
- Полный конвейер патчинга APK для обхода системы лицензий XDSDK
- Скрипт декомпиляции (`decompile.sh`)
- Скрипт патчинга native библиотеки (`patch_native.sh`)
- Скрипт патчинга Smali кода (`patch_smali.sh`)
- Скрипт сборки и подписи APK (`build.sh`)
- Единый скрипт для полного конвейера (`full_pipeline.sh`)
- Скрипт тестирования APK (`test_apk.sh`)
- Python скрипт для анализа native библиотек (`analyze_native.py`)

### Features
- Автоматический патчинг ARM64 native библиотек
  - Функция `check()` → всегда возвращает true
  - Функция `licence()` → всегда возвращает true
  - Сохранение функции `update()` для автообновлений
  - Сохранение алгоритма расшифровки burriEnc
  
- Патчинг Smali/DEX кода
  - SuperJNI класс - обход нативных проверок
  - LoginActivity - автоматический переход в MainActivity
  - Сохранение метода update для автообновлений
  
- Модификация AndroidManifest.xml
  - Изменение LAUNCHER activity с LoginActivity на MainActivity
  - Сохранение всех других intent-filters
  
- Сборка и подпись
  - Автоматическая очистка старых подписей
  - Создание нового keystore при необходимости
  - zipalign для оптимизации
  - Подпись V2/V3 для совместимости с Android 11+
  
- Тестирование
  - Автоматическая установка на устройство
  - Сбор и анализ логов
  - Проверка на ошибки и крэши

## Notes

### Breaking Changes
- Версии 1.x будут поддерживать текущую структуру XDSDK
- При изменении структуры XDSDK может потребоваться обновление скриптов
- Keystore формат может измениться в будущих версиях

### Migration Guide
N/A - первая версия

### Security
- Keystore файлы НЕ должны коммититься в репозиторий
- Используйте environment variables для паролей в CI/CD
- Регулярно обновляйте зависимости (apktool, Java)
- Проверяйте пропатченные APK на вирусы перед распространением

### Known Issues
- Автоматический патчинг native библиотеки может не работать если символы stripped
- В некоторых версиях XDSDK структура классов может отличаться
- radare2 патчинг может требовать установки дополнительных зависимостей
- На некоторых устройствах могут быть проблемы с V2/V3 подписями
